## 2. Facebook Pixel

### What is the Pixel?

It is a tracking code installed on your website to:

* Measure conversions (purchases, sign-ups, etc.)
* Optimize ad delivery
* Create remarketing audiences

### Select an Existing Pixel

1. In the **"Facebook Pixel"** field, click to open the list
2. You will see the pixels available in your ad account
3. Select the pixel installed on your website

### Create a New Pixel

If you don't have a pixel:

1. Click **"Create New Pixel"**
2. Enter a name (e.g., "My Store Pixel")
3. Click **"Create Pixel"**
4. The pixel is created and automatically selected

If you don't have a pixel, I can create one for you in seconds.

<Callout type="warn" title="Important">
  After creating the pixel, you must install it on your website. Check Meta's documentation for installation instructions.
</Callout>


### I Don't Have a Website

If you don't have a website, you can skip this field. Your ads will work normally, but you won't be able to measure web conversions.

## 5. Website URL

### What is it for?

Your website URL is used as:

* Default destination for your ads
* Domain for Pixel verification
* Suggested landing page

### Enter URL

1. In the **"Website URL"** field, type the address
2. You must include `https://` or `http://`
3. Example: `https://www.mystore.com`

### Validation Errors

| Error              | Cause               | Solution                  |
| ------------------ | ------------------- | ------------------------- |
| "Invalid URL"      | Incorrect format    | Include https://          |
| "Invalid protocol" | Doesn't start with http | Use https://example.com |

## Complete Setup

Once you have configured the assets you need:

1. Review the configuration summary
2. Click **"Next"** to go to the final review
3. On the review screen, verify everything is correct
4. Click **"Complete Setup"**

### Success Message

You will see: **"Meta Ads setup completed successfully!"**

Your status will change to **"Configured"** and you will be ready to create strategies.

## Edit Configuration Later

You can modify these assets at any time:

1. Go to **Connected Accounts**
2. In Meta Ads, click **"Edit Configuration"**
3. Modify the necessary assets
4. Save the changes
