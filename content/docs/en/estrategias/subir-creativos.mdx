If you prefer to use your own images or videos, or if the strategy requires videos (which cannot be generated with AI), here you'll learn how to upload them correctly to get the best results in your campaigns.

## Prepare Your Files Before Uploading

Before starting the upload, spend a few minutes preparing your files. A well-prepared creative makes the difference between an ad that goes unnoticed and one that generates clicks and conversions.

**Preparation checklist:**
- Verify that your files don't exceed 10 MB (compress beforehand if needed)
- Make sure they're in a supported format (MP4 for video, JPG/PNG for images)
- Check that the resolution is at least 1080px on its shortest side
- If it's a video, verify that the first 3 seconds capture attention
- If it's an image, make sure the product or message is clearly visible

**Free tools to prepare files:**
- **Videos:** HandBrake (to compress and convert to MP4)
- **Images:** TinyPNG (to compress without visible quality loss) or Canva (to resize)

## File Requirements

### For Videos

| Requirement                | Specification        |
| -------------------------- | -------------------- |
| **Formats**                | MP4, MOV, AVI        |
| **Maximum size**           | 10 MB per file       |
| **Recommended resolution** | 1080x1920 (vertical) |
| **Duration**               | 15-60 seconds ideal  |

### For Images

| Requirement                | Specification                          |
| -------------------------- | -------------------------------------- |
| **Formats**                | JPG, PNG                               |
| **Maximum size**           | 10 MB per file                         |
| **Recommended resolution** | 1080x1080 (square) or 1080x1350 (4:5) |

## File Characteristics

This section explains what your files should look like to get the best results on each platform and format.

### Videos for Instagram/Facebook

Videos are the format with the highest interaction rate on social media. If you prepare them correctly, they can significantly outperform static images.

**Recommended formats:**
- **Feed:** 1:1 (square) or 4:5 (vertical)
- **Stories/Reels:** 9:16 (full vertical)

**Best practices:**
- The first 3 seconds should grab attention -- if you don't hook viewers at the start, they'll keep scrolling
- Include subtitles (many watch without audio, especially on public transport or at the office)
- End with a clear call to action (visit our website, buy now, send a message)
- Use natural or well-controlled lighting; dark videos generate less engagement

### Images for Ads

Images are the most versatile and fastest format to produce. A good product image can generate excellent results if you follow these practices:

**Recommended formats:**
- **Feed:** 1:1 or 4:5
- **Stories:** 9:16

**Best practices:**
- Use vibrant colors and contrast to stand out in the feed
- Little text on the image (less than 20%) -- Meta penalizes ads with too much overlaid text
- Product or service clearly visible -- the user should understand what you're selling in under 2 seconds
- High resolution (minimum 1080px) -- blurry images destroy credibility

## How to Upload Files

The upload process is straightforward and designed so you can upload all your material in just a few minutes.

### 1. Access the Uploader

On the strategy configuration screen:

1. Find the **"Upload creatives"** section
2. Make sure you're on the **"Upload manually"** tab

You'll see how to upload images and videos according to what the strategy requires. Each strategy tells you the type and quantity of creatives it needs.

### 2. Select Files

You have two equally valid options:

**Option A: Drag and drop**

1. Open the folder with your files
2. Select the files
3. Drag them to the indicated area

**Option B: Click to select**

1. Click on the upload area
2. Navigate to your files
3. Select and confirm

<Callout type="info" title="Tip">
  You can select multiple files at once by holding down the Ctrl key (Windows) or Cmd key (Mac) while clicking each file.
</Callout>


You can use whichever method you prefer. Both work exactly the same.

### 3. Verify the Upload

After selecting your files, I show you a preview so you can confirm everything is correct:

* You'll see a list of uploaded files
* Each file shows: name, size, thumbnail
* The counter updates: "2/3 files"

I show you on screen which files you've uploaded and which ones are still missing so there's no doubt.

### 4. Manage Files

For each file you can perform the following actions if you need to make adjustments:

| Action           | How to do it                      |
| ---------------- | --------------------------------- |
| **Delete**       | Click the delete button           |
| **Reorder**      | Drag to change order              |
| **View options** | Click the menu (...)              |

The order of files matters: the first creative in the list will be shown as the main one in your ad.

## Number of Creatives

Each strategy indicates how many creatives it needs to work optimally:

```
Required: 3 videos
```

* **Minimum:** The indicated quantity
* **Maximum:** You can upload more if you want

<Callout type="info" title="Note">
  If you upload more than required, all of them will be used in the campaign. Meta will test each creative with your audience and show the best-performing one more often, which automatically optimizes your investment.
</Callout>

## When to Use Manual Upload vs AI

This table helps you decide when it's better to upload your own files and when it's better to let the AI generate the creatives:

| Situation                                     | Recommendation               |
| --------------------------------------------- | ---------------------------- |
| You have professional product photos           | Upload manually              |
| The strategy requires videos                   | Upload manually              |
| You don't have images ready                    | Generate with AI             |
| You want to test quickly                       | Generate with AI             |
| You need specific images of your product       | Upload product photo + AI    |

**Professional advertising, extreme simplicity.** No matter which path you choose: the result is always a professional-quality ad ready to convert.

## Common Errors

### "File too large"

**Message:** The file exceeds the maximum size of 10MB.

**Solution:**

1. Compress the video/image before uploading
2. Reduce the resolution (1080px is sufficient for social media)
3. Use a tool like HandBrake (videos) or TinyPNG (images)

### "Unsupported format"

**Message:** The file format is not valid.

**Solution:**

1. Convert to a supported format
2. Videos: use MP4 (it's the most compatible format with the best compression)
3. Images: use JPG or PNG

### "Upload error"

**Cause:** Connection issue or corrupted file.

**Solution:**

1. Check your internet connection
2. Try with a different file to rule out the original being corrupted
3. Reload the page and try again

Don't worry if a file fails: the other files you already uploaded are not lost. You only need to retry the one that failed.

## After Uploading

When you have all creatives uploaded and verified:

1. Verify that the counter shows "Ready"
2. Click **"Create ad"**
3. The files will be uploaded to Meta automatically
4. You'll proceed to the Preview screen where you can review how your ad will look before launching it
