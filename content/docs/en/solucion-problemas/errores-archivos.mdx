This guide covers the most common errors when uploading creatives and audio. Do not worry, most of them are resolved by adjusting the format or file size.

---

## Video and Image Errors

### "File too large"

**Message:** The file exceeds the maximum allowed size.

**Limit:** 10 MB per file

**Solution:**

1. Compress the file
2. Reduce the resolution
3. For videos: use HandBrake
4. For images: use TinyPNG or similar

The idea is to reduce the file size without losing quality.

---

### "Unsupported format"

**Message:** The file format is not valid.

**Valid formats:**

* Videos: MP4, MOV, AVI
* Images: JPG, PNG

**Solution:**

1. Convert to a supported format
2. Use free online converters
3. Avoid formats like HEIC, WEBP, GIF

You can convert your files to the correct format in minutes.

---

### "Upload error"

**Causes:**

* Unstable internet connection
* Corrupt file

**Solution:**

1. Check your connection
2. Try a different file
3. Reload the page

---

### "Error uploading videos to Meta"

**Message:** `Error uploading videos to Meta. Please try again.`

**Causes:**

* Connection issue with Meta
* File not compatible with Meta

**Solution:**

1. Verify the video meets Meta requirements
2. Try again
3. If it fails, use a different video

---

## Audio Errors (Brief)

### "Audio too short"

**Message:** `Please record at least 10 seconds of audio to provide enough context.`

**Requirement:** Minimum 10 seconds

**Solution:**

1. Record a longer audio
2. Answer all the guide questions
3. Provide more detail about your business

With a bit more context, the Brief comes out much better.

---

### "Maximum duration reached"

**Message:** `The recording stopped automatically upon reaching 3 minutes.`

**Limit:** Maximum 3 minutes

**Note:** This is not an error. The 3-minute audio is sufficient.

---

### "Audio file too large"

**Message:** `The audio file must not exceed 50MB.`

**Limit:** 50 MB for audio

**Solution:**

1. Compress the audio
2. Use a shorter file
3. Convert to MP3 with a lower bitrate

---

### "Audio too long"

**Message:** `The audio file must not exceed 3 minutes in duration.`

**Solution:**

1. Trim the audio to a maximum of 3 minutes
2. Summarize the most important information
3. Record directly on the platform

---

### "Invalid file"

**Message:** `Please select a valid audio file.`

**Valid formats:** MP3, WAV, M4A, OGG, WEBM

**Solution:**

1. Use a supported format
2. Verify the file is not corrupt
3. Try a different file

---

### "Could not access the microphone"

**Message:** `Please allow microphone access in your browser.`

**Solution:**

1. Click the lock icon in the address bar
2. Look for "Microphone"
3. Change to "Allow"
4. Reload the page

---

### "Error loading audio"

**Message:** `Could not process the audio file.`

**Causes:**

* Corrupt file
* Unsupported format
* Upload issue

**Solution:**

1. Try a different file
2. Record directly on the platform
3. Verify the file plays correctly

## File Requirements

### Videos

| Requirement            | Specification        |
| ---------------------- | -------------------- |
| Formats                | MP4, MOV, AVI        |
| Maximum size           | 10 MB                |
| Recommended resolution | 1080x1920 (vertical) |
| Duration               | 15-60 seconds        |

### Images

| Requirement            | Specification          |
| ---------------------- | ---------------------- |
| Formats                | JPG, PNG               |
| Maximum size           | 10 MB                  |
| Recommended resolution | 1080x1080 or 1080x1350 |

### Audio

| Requirement      | Specification            |
| ---------------- | ------------------------ |
| Formats          | MP3, WAV, M4A, OGG, WEBM |
| Maximum size     | 50 MB                    |
| Minimum duration | 10 seconds               |
| Maximum duration | 3 minutes                |
