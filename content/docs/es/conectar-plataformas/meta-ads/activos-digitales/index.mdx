## 2. Pixel de Facebook

### ¿Qué es el Pixel?

Es un código de seguimiento que se instala en tu sitio web para:

* Medir conversiones (compras, registros, etc.)
* Optimizar la entrega de anuncios
* Crear audiencias de remarketing

### Seleccionar Pixel Existente

1. En el campo **"Pixel de Facebook"**, haz clic para abrir la lista
2. Verás los pixels disponibles en tu cuenta publicitaria
3. Selecciona el pixel que está instalado en tu web

### Crear Nuevo Pixel

Si no tienes pixel:

1. Haz clic en **"Crear Nuevo Pixel"**
2. Ingresa un nombre (ej: "Pixel Mi Tienda")
3. Haz clic en **"Crear Pixel"**
4. El pixel se crea y queda seleccionado

Si no tienes pixel, yo puedo crearlo por ti en segundos.

<Callout type="warn" title="Importante">
  Después de crear el pixel, debes instalarlo en tu sitio web. Consulta la documentación de Meta para instrucciones de instalación.
</Callout>


### No Tengo Sitio Web

Si no tienes sitio web, puedes omitir este campo. Tus anuncios funcionarán normal, pero no podrás medir conversiones en web.

## 5. URL del Sitio Web

### ¿Para qué sirve?

La URL de tu sitio web se usa como:

* Destino predeterminado para tus anuncios
* Dominio para verificación de Pixel
* Página de destino sugerida

### Ingresar URL

1. En el campo **"URL del Sitio Web"**, escribe la dirección
2. Debes incluir `https://` o `http://`
3. Ejemplo: `https://www.mitienda.com`

### Errores de Validación

| Error                | Causa               | Solución                |
| -------------------- | ------------------- | ----------------------- |
| "URL inválida"       | Formato incorrecto  | Incluye https://        |
| "Protocolo inválido" | No empieza con http | Usa https://ejemplo.com |

## Completar Configuración

Una vez configurados los activos que necesitas:

1. Revisa el resumen de configuración
2. Haz clic en **"Siguiente"** para ir a la revisión final
3. En la pantalla de revisión, verifica que todo esté correcto
4. Haz clic en **"Completar Configuración"**

### Mensaje de Éxito

Verás: **"¡Configuración de Meta Ads completada exitosamente!"**

Tu estado cambiará a **"Configurado"** y estarás listo para crear estrategias.

## Editar Configuración Después

Puedes modificar estos activos en cualquier momento:

1. Ve a **Cuentas Conectadas**

3. Modifica los activos necesarios
4. Guarda los cambios
